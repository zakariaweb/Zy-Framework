# Zy-Framework

Zy-Framework is the home of ZyUI (CSS) and ZyJS (JS) for ZyFOCUS and other products.

## Structure

- `packages/zyui` - NPM package for ZyUI + ZyJS
- `docs` - ZyUI and ZyJS documentation

## Quick Start

```bash
cd packages/zyui
npm publish --access public
```
